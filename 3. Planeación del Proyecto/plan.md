## 2. Cronograma y gestión de tiempos

A partir del listado de actividades que conformarían en su totalidad el proyecto, se procedió a crear una línea del tiempo considerando los recursos humanos y físicos/digitales, así como aquellas tareas preparativas que permitieran cumplir con lo establecido en la EDT. Dada la naturaleza del proyecto, se decidió optar por una metodología de ejecución rápida.

Debido a las facilidades de integración entre GitHub Projects y el repositorio como entregable, se decidió crear un proyecto en esta misma plataforma. El cronograma establecido puede ser consultado en este [enlace](https://github.com/users/CVarPer/projects/3/views/2)
